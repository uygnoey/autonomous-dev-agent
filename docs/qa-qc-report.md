## 5. 경고 및 비크리티컬 이슈

### 5.1 pytest 경고 7건

pytest 실행 시 발생하는 경고 7건은 모두 비크리티컬입니다. 기능 결함이 아닌 deprecation 또는 테스트 설정 관련 경고이며, 현재 버전에서 동작에 영향을 주지 않습니다.

| 구분 | 건수 | 영향도 |
|------|------|--------|
| DeprecationWarning (라이브러리 내부) | 일부 | 없음 |
| 테스트 설정 경고 | 일부 | 없음 |
| **합계** | **7건** | **없음** |

### 5.2 스모크 테스트 자동화 제한

에이전트 실행 환경에서 `/dev/tty` 가 미구성되어 TUI 화면 자동화 테스트에 제한이 있습니다.
이는 에이전트 환경의 구조적 제약이며, 실제 사용자 환경에서는 정상 동작이 확인되었습니다.
